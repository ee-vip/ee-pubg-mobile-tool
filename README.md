# PUBG Mobile GFX Tool (Gameloop Emulator)

A lightweight tool designed to **optimize and customize PUBG Mobile graphics settings for the Gameloop emulator**.
It helps players unlock better performance, smoother gameplay, and improved visual settings directly from their PC.

## 🚀 Features

* Unlock higher **FPS options**
* Customize **graphics quality settings**
* Reduce **lag and frame drops**
* Improve **emulator performance**
* Simple and fast configuration
* Designed specifically for **Gameloop**

## 💻 Requirements

* Windows PC
* Gameloop Emulator installed
* PUBG Mobile installed through Gameloop

## ⚙️ How to Use

1. Launch the GFX Tool.
2. Select your desired graphics and FPS settings.
3. Apply the configuration.
4. Start PUBG Mobile through Gameloop and enjoy smoother gameplay.
